# 라우터가 하는 일
  ### 포워딩
  - 패킷 해더에서 목적지를 읽음
  - **포워딩 테이블**에서 그 목적지를 찾고 패킷을 보냄
  ### 라우팅
  - 포워딩 테이블을 만드는 것
  - 라우팅 알고리즘을 사용

### 인터페이스

- LAN카드, WIFI 어댑터 등...
- 한 컴퓨터에 인터페이스 n개 꼽으면 ip도 n개 할당 됨. 라우터가 그런식임
